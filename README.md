# Tic Tac Toe

A classic 3x3 grid game built with pure HTML, CSS, and JavaScript.

## How to Play

- Click on any cell to place your mark (X or O)
- Players take turns, starting with X
- First player to get 3 marks in a row (horizontal, vertical, or diagonal) wins
- If all cells are filled with no winner, it's a draw
- Click "Reset Game" to start over

## Features

- Turn-based gameplay
- Win detection with highlighting
- Draw detection
- Reset functionality
- Responsive design

## Tech Stack

- HTML5
- Vanilla JavaScript
- CSS3 Grid Layout
